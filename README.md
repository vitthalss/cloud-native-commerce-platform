## Overview

Enterprise-grade cloud-native microservices platform built using Spring Boot and Spring Cloud, demonstrating scalable distributed system architecture, secure service communication and modern DevOps deployment practices.

---

## Core Architecture Components

- API Gateway and centralized request routing
- Service Discovery using Eureka Server
- Inter-service communication using OpenFeign
- Reactive microservices with Spring WebFlux
- Distributed tracing and centralized logging using Zipkin and Sleuth
- Fault tolerance with Resilience4j circuit breakers
- Centralized configuration management using Spring Cloud Config
- OAuth2 and OpenID Connect based authentication and authorization
- Job scheduling and orchestration using Quartz Scheduler

---

## Cloud & DevOps Stack

- Docker containerization
- Kubernetes orchestration
- CI/CD pipeline integration
- Automated deployment workflows

---

## Data & Messaging Layer

- PostgreSQL and MySQL for transactional workloads
- Redis for distributed caching and high-performance access

---

## Testing & Quality

- JUnit
- Mockito
- WireMock
- AssertJ

---

## Purpose

This repository demonstrates enterprise microservices architecture patterns commonly used in scalable cloud-native commerce and distributed platform engineering environments.
